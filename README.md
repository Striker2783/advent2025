# AoC 2025 Workshop

Advent of Code (AoC) is an advent calendar for computer science problems found at https://adventofcode.com/2025.

# How to Run

1. Install the Rust Programming Language https://rust-lang.org/learn/get-started/.
2. Clone this repository.

```
git clone https://github.com/Porogaramu/aoc2025.git
```

3. Go into the directory
```
cd aoc2025
```
4. Run the program. Day is the day number to run with. The input file is the file AoC gives you.
```
cargo run --release <DAY> <INPUT_FILE>
```
